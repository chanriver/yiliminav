# yiliminav 开发任务分解

## 项目概述

基于 ModernNav 增强的 AI 导航网站，使用 React 19 + Tailwind CSS + Cloudflare (Pages/D1/KV/R2) + MiniMax-M2.5

### 详细说明

#### 1.1 Fork 并克隆项目
- 在 GitHub fork `chanriver/ModernNav`（或 `lyan0220/ModernNav`）
- 克隆到本地：`git clone <your-fork-url> yiliminav`
- 或者直接复制 `ModernNav-修改后` 目录内容

#### 1.2 安装依赖
```bash
cd yiliminav
npm install
```

#### 1.3 Cloudflare 配置
需要创建：
- **D1 数据库**: 用于存储书签、用户数据
- **KV 命名空间**: 用于缓存名言、配置等
- **R2 存储桶**: 用于存储 AI 生成的封面图

#### 1.4 环境变量
```bash
# .env.local (开发用)
MINIMAX_API_KEY=your_api_key_here

# wrangler.toml (部署用)
# 通过 wrangler secret put MINIMAX_API_KEY 设置
```

**诗词 API**:

- 本地诗词库备选

**逐字显示实现**:
```tsx
const PoemWidget: React.FC = () => {
  const [poem, setPoem] = useState('');
  const [displayedText, setDisplayedText] = useState('');

  useEffect(() => {
    // 打字机效果
    let index = 0;
    const timer = setInterval(() => {
      if (index < poem.length) {
        setDisplayedText(poem.slice(0, index + 1));
        index++;
      } else {
        clearInterval(timer);
      }
    }, 100);
    return () => clearInterval(timer);
  }, [poem]);
};
```

**MiniMax API 调用示例**:
```typescript
// functions/api/recommend.ts
export async function onRequestPost(context: Context) {
  const { request, env } = context;
  const { MINIMAX_API_KEY } = env;
  
  const body = await request.json();
  const { query, categories } = body;

  const prompt = `用户需求: ${query}
可用分类: ${categories.join(', ')}
请推荐最合适的网站分类。`;

  const response = await fetch('https://api.minimax.chat/v1/text/chatcompletion_v2', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${MINIMAX_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      model: 'MiniMax-M2.5',
      messages: [{ role: 'user', content: prompt }],
      stream: false,
    }),
  });

  const result = await response.json();
  return new Response(JSON.stringify(result), {
    headers: { 'Content-Type': 'application/json' },
  });
}
```
